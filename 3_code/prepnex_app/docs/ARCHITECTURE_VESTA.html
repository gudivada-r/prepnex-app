Platform Core Components

A. Student Pathfinder Ecosystem (Ingestion)

Ingests specific 6-12th grade college competitive data:

  • Academic History: 6th–12th grade course rigor and GPA.
  • Competition Records: USACO, AMC, FRC, and Olympiads.
  • Participant Behavioral Data: Mentorship directives and coaching session history.

B. PrepNex Alignment Engine (Platform Layer)

The middle layer in the aumtech.ai ecosystem:

  • PostgreSQL (prepnex schema): Isolated high-school data namespace.
  • 7-Year Roadmap Compiler: Logic that projects current trajectories into senior-year elite targets.

C. Vesta Intelligence Layer

  • Vesta Privacy Gateway: Strips PII before bridging to Google Gemini.
  • Vesta Context Engine: Synthesizes the roadmap into proactive "Strategy Nudges."
  • The Bridge: Seamlessly transitions refined 12th-grade data into the aumtech.ai Aura (College) ecosystem.
